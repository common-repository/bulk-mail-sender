== Description ==

**Bulk Mail Sender** 
Bulk Mail Sender is use to send mail to the all user

#### Features:
1. Send mail to multiple users.
2. Manage users
3. Manage NewsLetters
4. Manage Templates
5. Analytics of sent mails
6. History
7. HTML Template Supportable
